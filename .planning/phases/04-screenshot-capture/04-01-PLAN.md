    Step 1 — Extend AppError (src/error.rs):
    Add two variants to the AppError enum:
    ```rust
    /// Window capture failed (non-permission error)
    #[error("Capture failed: {0}")]
    CaptureFailed(String),

    /// Screen Recording permission not granted (macOS)
    #[error("Screen Recording permission required.\nGo to: System Preferences > Privacy & Security > Screen Recording\nEnable access for this terminal application, then retry.\n(Original error: {0})")]
    PermissionDenied(String),
    ```

    Add constructor methods following the existing pattern:
    ```rust
    pub fn capture_failed(msg: impl Into<String>) -> Self { Self::CaptureFailed(msg.into()) }
    pub fn permission_denied(msg: impl Into<String>) -> Self { Self::PermissionDenied(msg.into()) }
    ```

    Step 2 — Create capture_service.rs (src/capture_service.rs):
    Write tests first (RED), then implement (GREEN).

    The `capture_window` function must:
    1. Call `xcap::Window::all()`, map error to anyhow with context "Failed to enumerate windows for capture"
    2. Find matching xcap Window: try `xcap_win.id().unwrap_or(0) as u64 == info.window_id` first.
       If no ID match, fall back to matching title AND pid simultaneously:
       `xcap_win.title().ok().as_deref() == Some(&info.title) && xcap_win.pid().ok() == Some(info.pid)`
       Add a comment: "ID correlation: xcap returns u32, WindowInfo stores u64; fallback to title+pid if no ID match"
    3. If no window found after both strategies, return `Err(anyhow!(AppError::capture_failed(...)))`
    4. Check `xcap_win.is_minimized().unwrap_or(false)`: if true, return error
       "Cannot capture minimized window — restore the window and retry"
    5. Match on `xcap_win.capture_image()`:
       - Ok(image) => proceed to save
       - Err(e) => check if error message (lowercased) contains "permission", "screen recording",
         "not permitted", or "access denied". If yes, return AppError::permission_denied(e.to_string()).
         Otherwise return AppError::capture_failed(e.to_string()).
    6. Before saving: create parent directories if output has a non-empty parent:
       `if let Some(parent) = output.parent() { if !parent.as_os_str().is_empty() { std::fs::create_dir_all(parent)?; } }`
    7. Call `image.save(output).with_context(|| format!("Failed to write PNG to {}", output.display()))?`
    8. Return Ok(())

    Add `pub mod capture_service;` to src/lib.rs.

    `cargo test --lib capture_service` must pass.
    `cargo test --lib` (all lib tests) must pass.
